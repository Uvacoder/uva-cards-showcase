![Preview](/assets/preview.png)

## Installation & Setup

1. Install dependencies

```sh
npm install
```

2. Start the development server

```sh
npm run dev
```

3. Generate production build

```sh
npm run build
```

4. Preview the production

```sh
npm run preview
```
